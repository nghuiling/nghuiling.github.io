Meal Planner

A reinforcement learning approach for meal planning based on budget and nutritional constraints

Project Overview

This project uses reinforcement learning to generate optimal meal plans based on user constraints. 5,485 meals were scraped from the web, each with a recipe, nutrition values, and manually added cost information. Users specify how many meals they want, their budget, and nutrition thresholds, and the model generates an ideal meal plan to meet these constraints.

Technologies Used

Project Details

The Challenge

Creating meal plans that satisfy multiple constraints like budget limits and nutritional requirements is a complex optimisation problem that is difficult and time-consuming to solve manually.

The Solution

I implemented a reinforcement learning model using Markov Decision Process (MDP) and Monte Carlo learning, where each state represents a collection of meals in a plan, and the final state is the optimised meal plan that best meets user constraints without any hassle.

Key Features
Dataset of 5,485 meals with recipes, nutrition values, and costs
User input for meal quantity, budget constraints, and nutrition thresholds
Reinforcement learning optimisation with MDP and Monte Carlo methods
Optimised meal plans that balance nutrition and budget constraints
