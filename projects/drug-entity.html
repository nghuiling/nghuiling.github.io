Drug Entity Recognition

A simple Flask application to recognise drugs from text using the Med7 model

Project Overview

This project uses the Med7 model for clinical named-entity recognition (NER) tasks. The model is trained on MIMIC-III free-text electronic health records and is able to recognise 7 categories of medical entities.

Technologies Used

Project Details

The Challenge

Medical professionals spend significant time manually extracting drug information from various documents, which is time-consuming and prone to human error.

The Solution

This Flask application automates the extraction of drug entities from text using the Med7 model, significantly reducing manual work and potential errors.

Entity Categories
DRUG - Names of drugs
STRENGTH - Strength of medications
FORM - Form of medication
FREQUENCY - How often the medication is taken
DOSAGE - Dosage of medication
ROUTE - Route of administration
DURATION - Duration of medication course
