Catching Snowballs

A fun mathematical game for elementary students built with Scratch

Completion Date: 4 Dec 2015
Duration: 5 Hours

Project Overview

Catching Snowballs is an educational game designed to help elementary students practice basic math skills in a fun, engaging environment. Players catch falling snowballs with numbers that match specific mathematical criteria.

Technologies Used

Project Details

The Challenge

Elementary students often find math practice repetitive and boring, which can lead to disengagement and difficulty retaining mathematical concepts.

The Solution

Catching Snowballs transforms math practice into an engaging game where students collect falling snowballs based on various mathematical conditions, making learning fun and interactive.

Key Features
Number categorisation challenges (more than, less than, between)
Colorful, winter-themed graphics to engage young learners
Real-time scoring system and achievement tracking
Sound effects and visual feedback for correct/incorrect answers
