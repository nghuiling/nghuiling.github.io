Daily Weather Forecast

Daily weather forecast web application using OpenWeather API

Project Overview

This web application provides users with daily weather forecasts using the OpenWeather API. Users can search for weather information by city name.

Technologies Used

Project Details

The Challenge

Creating a user-friendly interface to display weather data that updates in real-time and is accessible across various devices.

The Solution

This Django-based application integrates with the OpenWeather API to fetch real-time weather data and present it in a clean, responsive interface.

Key Features
Current weather conditions display
Location search functionality
Responsive design for all devices
