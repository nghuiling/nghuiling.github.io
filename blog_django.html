#1: How to build a Django App

Date: 6 Jan 2021

Step 1: Set up Virtual Environment using Git Bash

> cd c/user/user/... (cd to project folder directory)
> ls
 > pip freeze
 > python -m -venv venv
 > cd..
 > source venv/Scripts/activate
 > pip freeze (check if nothing shows up)
> clear

Step 2: Install Django

> pip install django
> django-admin.py startproject NAME1
> cd NAME1
> python manage.py runserver
Go to 'http://localhost:8000/' (check if working)


Step 3: Set up admin users and migrate database

> python manage.py migrate
> winpty python manage.py createsuperuser (fill in user and password)
Go to 'http://localhost:8000/admin' (log in to check if working)


Step 4: Create a new application

> python manage.py startapp NAME2
Go back to previous main directory (NAME1)
Go to 'settings.py' > Scroll to INSTALLED_APPS > Add 'NAME2'
Copy 'urls.py' file in NAME1 and paste in NAME2
Modify the 'urls.py' file in NAME1 to include NAME2:
  
from django.urls import path, include

path('', include('NAME2.urls')),
Remove "..import admin...' and 'path('admin/'....)' in 'urls.py' file in NAME2
Create a new folder 'templates' in NAME2 > Add 'home.html' file inside
In 'views.py' in NAME2, add:
  
def home(request):
  return render(request,'home.html',{})
In 'urls.py' in NAME2, add:
  
from . import views

urlpatterns = [
  path('',views.home, name='home'),
]
In 'templates' folder, create a new file 'base.html' and add:
  
HEADER
#Add Bootstrap

{% block content %}

{% endblock %}

FOOTER
#Add Bootstrap
Modify 'home.html':
  
{% extends 'base.html' %}

{% block content %}

Hello world!

{% endblock %}
Locate directory to NAME1 and run programme
> python manage.py runserver

Finally, go to 'http://localhost:8000/' on your browser to view: Hello world!
