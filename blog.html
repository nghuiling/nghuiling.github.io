How to build a Flask App

Date: 5 May 2020

Step 1: Ensure you have Python on your system environment variables

I added the following to my system environment variables path:
C:\Users\User\anaconda3
C:\Users\User\anaconda3\Scripts
C:\Users\User\anaconda3\Library\bin
C:\Users\User\anaconda3\Library\mingw-w64\bin

 To test the above, in Command Prompt:
> cd C:\Users\User
> python --version
Python version should appear.


Step 2: Install a virtual environment so that your installations for each project will be contained in that environment

In Command Prompt:
> pip install virtualenv


Step 3: Create a folder (e.g. named 'test' on your Desktop) for your app and build a virtual environment (e.g. named 'env')

In Command Prompt:
> cd C:\Users\User\Desktop\test
> virtualenv env


Step 4: Activate your 'env' environment and start installing Flask (and other things you want)

In Command Prompt:
> cd C:\Users\User\Desktop\test\env\Scripts
> activate.bat
> pip install Flask


Step 5: Create an app.py file in the 'test' folder

Insert the following in the app.py file:
from flask import Flask, render_template

app = Flask(__name__)

@app.route('/')

def index():
 return "Hello!!!!"

if __name__ =="__main__":
 app.run(debug=True)


Step 6: Run your app!

In Command Prompt:
> cd C:\Users\User\Desktop\test
> python app.py
Go to http://127.0.0.1:5000/ on your browser to view: Hello!!!!


Best practices for Flask app folder's directory:
> env
> static>css>main.css
> static>js
> templates>index.html
> app.py

This is just the beginning of building a Flask app, many more to come!
