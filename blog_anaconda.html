#1: How to set up a new conda environment

Date: 6 Jan 2021

Open Anaconda Prompt and type the following: 
  
conda create -n NAME python=VERSION
conda activate NAME

pip install ipykernel
python -m ipykernel install --user --name NAME --display-name NAME

#pip install your items#
For standalone, install individual files (.whl or .tar.gz) in a location (e.g. D:\Packages) and type the following in Anaconda Prompt: 
  
conda create --prefix=C:\ProgramData\Anaconda3\envs\NAME --clone root --offline
pip install NAME OF PACKAGE --no-index -f D:\Packages
