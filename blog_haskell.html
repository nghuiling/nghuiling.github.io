#1: How to set up Probabilistic Functional Programming (PFP) library for Haskell

Date: 27 May 2021

1. Use Chocolatey to install ghc and cabal-install
2. Install Stack at haskellstack.org
3. Download probability-0.2.7.tar.gz from hackage.haskell.org
4. Unzip probability-0.2.7.tar.gz into probability-0.2.7 folder
5. Using Comd Prompt:
  • change directory to probability-0.2.7 folder
  • stack init -> stack setup -> stack build -> stack ghci (folder_name).hs -> import (module_name)
